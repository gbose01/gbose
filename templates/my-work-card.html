Technology

Self-hosted Home Raspberry Pi Server running Linux and Docker (2022)

I spent a good month obsessed with learning about the raspberry pi and it's potential DIY applications. I settled on what I thought would be incredibly fun - turning it into a low-power mini-server in my apartment that hosts media content, among other things. I installed the Raspberry Pi OS (Linux Debian 11) and then used Docker / Portainer to install apps like Plex. I also enabled a file browser and a VPN server which are handy tools for my home network.
